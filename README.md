# PROJEKT: Nowoczesna Stodoła na Skarpie - Masterplan (Wersja 4.0)

## Opis projektu

Projekt dotyczy budowy nowoczesnej stodoły na skarpie, która ma na celu połączenie komfortu i estetyki z efektywnością energetyczną. Kluczowe aspekty projektu obejmują:

- **Specyfikacja działki**: szerokość 19.8m, nachylenie 2.5m
- **Rodzaj budynku**: jednopiętrowy dom o powierzchni wewnętrznej 141 m² oraz dodatkowej przestrzeni na antresoli o powierzchni 30 m²
- **Fundamenty**: budowa na fundamentach schodkowych
- **Standard energetyczny**: standard pasywny/premium
